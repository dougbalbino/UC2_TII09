Introdução à tag < img >

A tag img em HTML é usada para incorporar imagens em uma página web. Ela é um elemento vazio, ou seja, não possui uma de fechamento e contém apenas atributos que definem o conteúdo visual.

Atributos principais

  • src (source): define o caminho da imagem que será exibida. Pode ser uma URL externa ou um caminho relativo dentro do servidor.

    Exemplo 1: 

    < img src="./Imagem1.png" >

    Resultado:

  • alt (alternative text): Fornece um texto alternativo para ser exibido caso a imagem não possa ser carregada. Ele também melhora a acessibilidade para leitores de tela.

    Exemplo 2:

    < img src="./Blaster e Soul.gif" alt="Black Star e Soul Correndo" >

  • src também pode ser de um link externo.

    Exemplo 3:

     < img src="https://filepicker-images.genius.com/f1g4006yu2" alt="Lua Sangrenta >

Tipos Comuns de Imagens

Os formatos de imagens mais utilizados na web são:

  • (.jpg ou .jpge) Formato utilizado amplamente para fotografias e imagens com muitos detalhes e cores. Possui compreesão com perda de qualidade (lossy).
  • (.png) Usado para imagens com fundo transparente ou que necessitam de alta qualidade, sem perda de detalhes. Utiliza compreesão sem perda (lossless).
  • (.gif) Formato ideal para imagens animadas e pequenos icones. Suporta transparência mas tem um número limitado de cores.
