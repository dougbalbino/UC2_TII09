Síntese das Propriedades

1.color

Define a cor do texto de um elemento.

h1 {
    color: blue;
}

2.text-align

Alinha o texto dentro de um elemento.

  • left: alinha o texto à esquerda (padrão).
  • center: centraliza o texto.
  • right: alinha o texto à direita
            h1 {
                text-align: center;
            }

3.font-size

Define o tamanho da fonte de um elemento.

            p {
                font-seize: 18px;
            }

4.background-color

Define a cor de fundo de um elemento.

            button {
                background-color: yellow;
            }

color(para botões)

Define a cor do texto de um elemento (usado em conjunto com background-color).

            button {
                color: black;
            }
