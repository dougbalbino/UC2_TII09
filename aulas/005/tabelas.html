Introdução às Tabelas em HTML

Tabelas são estruturadas em HTML que permitem a organização de dados em linhas e colunas. São comumente usadas para exibir calendários, horários, listas de preços, etc.

Estrutura básica de uma tabela

A estrutura básica de uma HTML é composta por várias tags:

  • table: Define o ínicio e fim de uma tabela.
  • tr: Define uma linha na tabela.
  • td: Define uma célula dentro de uma linha (dados).
  • th: Define uma célula de cabeçalho.

Exemplo 1: Tabela Básica

Nome Idade Profissão
Ana 28 Engenheira
Bruno 34 Médico

Explicação

  • Nesse exemplo usamos th para o cabeçalho e td para os dados.
  • O atributo border="1" é usado para adicionar bordas à tebela

Mesclagem de células

A mesclagem de células pode ser feita tanto em colunas quanto em linhas.

  • colspan: Mescla células horinzontalmente
  • rowlspan: Mescla células verticalmente
Nome Idade Profissão
Carlos 30 Professor
31 Arquiteto
Daniela Designer Gráfico

Explicação

  • rowspan ="2": A célula "Carlos" se expande verticalmente para cobrir duas linhas.
  • colspan ="2": A célula "Designer Gráfico" se expande horizntalmente para cobrir duas colunas.

Adicionando um cabeçalho e rodapé à tabela

Além de linhas de dados, uma tabela pode ter um cabeçalho (thead), um corpo (tbody) e um rodapé (tfoot).

Produto Quantidade Preço Unitário
Caneta 10 R$ 2,00
Caderno 5 R$ 15,00
Total: R$ 95,00

Explicação:

  • thead: Define o cabeçalho da tabela
  • tbody: Define o corpo da tabela, onde ficam os dados
  • tfoot: Define o rodapé da tabela, ideal para somatórios ou resumos

    • Tabela Aninhada
